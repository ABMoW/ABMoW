Frequently Asked Questions

Man O'War Data Files

Welcome to the FAQ for the Man O'War data files for Army Builder. This FAQ contains a collection of important details about the data files for this game system that their author thought you should be aware of. If you have questions that are not covered below, please contact the author of the data files for further information. You will find contact information for the author under the 'Help' menu by selecting the 'Contact Data File Author' menu item.

The data file author chose not to include FAQ entries. Please contact the data file author at 'malkithe@hotmail.com' if you have any questions.
